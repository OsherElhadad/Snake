# Snake Stages

אפליקציית Snake עצמאית בדפדפן עם:

- 3 שלבים עם קושי עולה
- מכשולים בשלבים מתקדמים
- מד התקדמות לכל שלב
- עצירה, אתחול, ניצחון והפסד
- תמיכה במקלדת עם חצים או `WASD`

## הרצה

אפשר לפתוח את `index.html` ישירות בדפדפן, או להריץ שרת מקומי:

```bash
cd /Users/osherelhadad/Documents/snake-stages
python3 -m http.server 8000
```

ואז להיכנס לכתובת:

`http://localhost:8000`
